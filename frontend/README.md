# Morsel Frontend — App Layer 📱

This is the Next.js 16 (App Router) frontend for Morsel.

## Tech
- **Next.js 16**: High-performance SSR and Server Components.
- **Tailwind CSS v4**: Modern, utility-first styling.
- **Lucide-React**: Vector iconography.
- **Recharts**: Responsive biometric data visualization.

## Development
```bash
npm install
npm run dev
```

## Production
```bash
npm run build
npm run start
```

For full system setup, see the root [README](../README.md).
